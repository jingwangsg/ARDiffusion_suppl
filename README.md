## Insights from Error Term Decompositions

<center><img src="assets/fvd_comparison_timestep_scheduling.png" alt="Timestep Scheduling" width="1000"/></center>
<center><div style="width: 800px">Fig 2. Error Accumulation using Different Timestep Scheduling. </div></center>

Fig 2. demonstrates the error accumulation of OpenSoraPlan+FIFO with different time steps schedulers. We consider the schedule that is $\text{Uniform}([0,999^{\alpha}] )^{1/\alpha}$. Here we set $\alpha$ as $1/2$ (order 2) and $1/3$ (order 3). The ``reverse'' refers to the schedule that is $999 - \text{Uniform}([0,999^{\alpha}] )^{1/\alpha}$
